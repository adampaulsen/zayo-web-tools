What is SFAST?

SFAST (SalesForce Automation Search Tool) is a web application that helps you generate multiple SalesForce search URLs from a list of identifiers or data values.

Instead of manually creating search URLs one by one, you can input multiple values and generate them all at once, then open them in new browser tabs for efficient searching.

Example Use Cases

Sales Teams

Search multiple customer records, opportunity IDs, or account numbers simultaneously.

Support Staff

Look up multiple case numbers, ticket IDs, or customer reference numbers.

Administrators

Process bulk operations on user accounts, system records, or configuration items.

Data Analysts

Query large datasets by generating search URLs for multiple data points.

How to Use SFAST

Prepare Your Data: Gather the identifiers or values you want to search for in SalesForce. You can use any format - DFIDs, case numbers, account numbers, etc.
Input Your Data: Copy and paste your data into the text area. You can separate values with commas, new lines, or both. SFAST will automatically parse and clean the input.
Generate URLs: Click the "Generate URLs" button. SFAST will create a table showing all the generated SalesForce search URLs.
Review Results: Check the generated URLs to ensure they're correct. Each URL will open the SalesForce search page for that specific value.
Open URLs: Use "Open All URLs" to launch all searches in new browser tabs, or click individual links to open specific searches.
Clear When Done: Use "Clear Results" to reset the tool and start fresh with new data.

Keyboard Shortcuts

Power users can use these keyboard shortcuts for faster operation:

  • Ctrl + Enter - Generate URLs
  • Ctrl + K - Focus input field
  • Ctrl + O - Open all URLs
  • Ctrl + C - Clear results

Tips for Best Results

  • Clean Your Data: Remove any unnecessary characters or formatting before pasting
  • Batch Processing: Process data in batches of 50-100 items for optimal performance
  • URL Verification: Always verify a few generated URLs before opening all of them
  • Browser Tabs: Be mindful of how many tabs you open - very large numbers may slow down your browser
  • SalesForce Access: Ensure you have proper access to the SalesForce instance you're searching
